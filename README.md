Решены следующие задачи

## Наследование
Класс Mentor - родительский класс (реализованы Фаимилия, Имя, Отчество и список закрепленных курсов), от него наследуются классы Lecturer и Reviewer.

## Атрибуты и взаимодействие классов
Только Reviewer может выставлять оценки студентам за ДЗ, а лекторам за проведенные лекции. Реализован метод выставления оценок лекторам у класса Student (оценки хранятся в атрибуте-словаре у Lecturer, где ключи - названия курсов, а значения - списки оценок). Лектор закрепляется за курсом, на который записан студент.

## Полиморфизм и магические методы
Перегружен метод ```__str__```  у всех классов.

У Reviewer выводится информация в следующем виде:
```python
print(reviewer_1)
Фамилия: Петров
Имя: Петр
Отчество: Петрович
```
У Lecturer:
```python
print(lecturer_1)
Фамилия: Иванов
Имя: Иван
Отчество: Иванович
Средняя оценка за лекции: 10.00
```
У Student:
```python
print(student_1)
Фамилия: Колотушкина
Имя: Александра
Отчество: Федоровна
Средняя оценка за ДЗ: 5.66
Курсы в процессе изучения: Python
Завершенные курсы: Java
```

### Реализована возможность сравнения между собой лекторов по средней оценке за лекции и студентов по средней оценке за ДЗ

## Полевые испытания
Созданы экземпляры классов, вызваны все созданные методы и реализованы две функции:

1. Подсчет средней оценки за ДЗ по всем сдуентам в рамках конкретного курса;
2. Подсчет средней оценки за лекции всех лекторов в рамках курса.
